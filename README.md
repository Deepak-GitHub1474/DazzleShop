# E-Commerce Application DazzleShop

### Setup instructions:

1. Clone the project
    ```
        git clone https://github.com/Deepak-GitHub1474/DazzleShop.git
    ```
2. Move into the directory
    ```
        cd DazzleShop
    ```
3. Install dependencies
    ```
        npm i
    ```

4. Run the server
    ```
        npm run dev
    ```
